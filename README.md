# 🚀 AI Resume Analyzer








An AI-powered Resume Analyzer that evaluates resumes and provides ATS scores, feedback, and improvement suggestions to help job seekers create stronger resumes and improve their chances of getting shortlisted.

This project uses AI analysis + PDF parsing + modern frontend technologies to provide an intelligent resume evaluation system.
